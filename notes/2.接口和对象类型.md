#### 接口 interface
```js
1、重复声明会合并
2、可选操作符 ?
3、联合类型 | 
4、任意类型 [propName: string]: any
5、只读 readonly 
6、函数属性
7、继承 interface A extends B
    interface A{
       readonly name: string
    } 
    // 重复声明会合并
    interface A{
        age?: number
        [propName: string]: any // 任意属性
        cb(): void
    }
    let obj:A = {
        name: ''
    }
```
